# Job Application Tracker
A web application to help track job applications during the job
search process.
## Features
- Track companies and job listings
- Record application submissions
- Manage interview schedules
- Store contact information
## Technologies
- MySQL Database
- Python with Flask
- HTML/CSS for the web interface

# Usage Guide 
Step 1. Download all of the files to your device (Mac or PC works) 
Step 2. Open a Command Prompt connection and navigate to the location of the job_tracker folder (for example C:\Users\Giovanna\Downloads\job_tracker\job_tracker) 
Step 3. Open a MySQL connection by typing "py app.py" in the Command prompt 
Step 4. Once the connection is open, launch a browser tab (Chome or Google will suffice) and type in the following address: http://localhost:5000/
Step 5. Have fun exploring the web application!
